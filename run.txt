Opcion 1: Live Server

Abrir index.html con Live Server desde la carpeta raiz del proyecto:

P3PracticaRopa_IA

Opcion 2: Terminal

Ejecutar este comando dentro de la carpeta P3PracticaRopa_IA:

python3 -m http.server 8000

Luego abrir en el navegador:

http://localhost:8000

Nota:
No hace falta backend. Solo se necesita servir los archivos estaticos para que el navegador pueda cargar:

tfjs_target_dir/model.json
tfjs_target_dir/group1-shard1of1.bin
